# SerfdomCraft
## What is it about?
Mod consists of three main parts:
*  Medieval combat system
*  Resource-oriented economy
*  Convoy system

## Prerequisities
* **Minecraft** -- *1.12.2 only*
* **Either**:
  * **Minecraft forge** -- *1.12.2-2705 and above* 
  * **Optifine**  --  *1.12.2 - any version*
